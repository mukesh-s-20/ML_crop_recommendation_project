What This System Does

🔬

Preprocessing

IQR-based outlier capping, StandardScaler normalization, Label Encoding, and 75/25 stratified split.

🤖

Classification

Decision Tree, Random Forest, KNN, Naive Bayes, SVM, Logistic Regression – all compared side-by-side.

🔭

Clustering

K-Means, DBSCAN, Hierarchical (Ward), and Apriori-style pattern clustering for unsupervised insights.

🧠

Deep Learning

3-layer MLP neural network (128→64→32 neurons) with ReLU, Adam optimizer, and L2 regularisation.

🔗

Association Rules

Apriori algorithm from scratch to mine frequent soil-condition ↔ crop patterns with Support, Confidence, Lift.

🌐

Web Interface

Full-stack Flask web app with real-time prediction, interactive charts, and model comparison dashboard.
