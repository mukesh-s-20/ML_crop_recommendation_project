📖 About This Project

AI-Based Crop Recommendation System – Final Year ML Project

Problem Statement

Farmers in India often rely on traditional knowledge or trial-and-error to decide which crop to grow. Poor crop selection leads to low yields, wasted resources, and financial loss. This project proposes an AI-based crop recommendation system that uses soil parameters and climate data to intelligently suggest the most suitable crop for a given field.

Dataset Details

  • 📁 Source: Crop Recommendation Dataset (Excel)
  • 📊 Total Samples: 2,200 rows × 8 columns
  • 🌾 Target Classes: 22 unique crop types
  • ⚖️ Balanced: 100 samples per crop
  • ✅ No missing values | 0 duplicates after cleaning
NitrogenPhosphorusPotassium TemperatureHumiditypH ValueRainfall

Technical Pipeline

1

Preprocessing

Duplicate removal, IQR-based outlier capping, StandardScaler normalization, LabelEncoder, 75/25 stratified split

2

Classification

Decision Tree, Random Forest, KNN, Naive Bayes, SVM, Logistic Regression — with Accuracy, Precision, Recall, F1, CV Score

3

Clustering

K-Means (Elbow method), DBSCAN (density-based), Hierarchical/Ward (Dendrogram), Apriori-pattern clustering

4

Deep Learning

3-layer MLP Neural Network (128→64→32), ReLU activation, Adam optimizer, L2 regularisation, Early Stopping, ROC-AUC

5

Association Rules

Apriori algorithm from scratch: itemset generation, rule extraction, Support/Confidence/Lift evaluation

6

Web App

Flask REST API with real-time prediction, interactive dashboard, responsive UI with chart gallery
